Git命令

Author: 陈士方 Date: 三月 2, 2020  14:49:18 Category: Git

全局设置
git config –global user.name “陈士方”
git config –global user.email “5695094+chenshifang@user.noreply.gitee.com“

创建仓库
mkdir test

进去仓库
cd test

初始化项目
git init

linux系统下用（windows系统可省略）
touch README.md

提交本地仓库（添加暂存区到区）
git add README.md/.（点代表所有）

从暂存区提交（-m:描述）
git commit -m “first commit”

把线上仓库和本地仓库连接（只需连接一次即可）
git remote add origin https://gitee.com/chenshifang/test.git

推送到线上master分支
git push -u origin master

清掉配置
git config –system –unset credential.helper

保存密码
git config –global credential.helper store

克隆项目
git clone https://gitee.com/chenshifang/DjangoPro.git

回退（没有git add时用）
git checkout .

回退（git add后用）
git reset HEAD

回退到某个版本
git reset –hard 日志号

打印提交日志
git log

打印所有标签
git tag

将指定的标签提交到远程仓库
git push <远程仓库的别名> <标签名称>

将本地所有的标签全部提交到远程仓库
git push <远程仓库的别名> –tags

删除指定标签
git tag -d 标签名

删除指定文件夹，并从本地仓库的文件夹中删除
git rm（要删除的文件夹路径）

删除指定的文件夹，在本地库的文件夹中保留该文件
git rm –cached（要删除的文件夹路径）

素质三连
git add .
git commit -m ‘描述’
git push -u origin master
