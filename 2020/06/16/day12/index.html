Linux-安装MySQL、jdk

Author: 陈士方 Date: 六月 16, 2020  22:21:18 Category: 测试笔记

Linux下安装jdk

1，创建一个空文件夹

mkdir jdk

2，进入jdk文件夹

cd jdk

3，下载jdk 

wget --no-check-certificate --no-cookies --header "Cookie: oraclelicense=accept-securebackup-cookie" http://download.oracle.com/otn-pub/java/jdk/8u131-b11/d54c1d3a095b4ff2b6607d096fa80163/jdk-8u131-linux-x64.rpm


4，解压安装

chmod +x jdk-8u131-linux-x64.rpm
rpm -ivh jdk-8u131-linux-x64.rpm

5，修改配置文件 

cd /
cd etc
vi profile

export JAVA_HOME=/usr/java/jdk1.8.0_131
export JRE_HOME=${JAVA_HOME}/jre
export CLASSPATH=.:${JAVA_HOME}/lib:${JRE_HOME}/lib:$CLASSPATH
export JAVA_PATH=${JAVA_HOME}/bin:${JRE_HOME}/bin
export PATH=$PATH:${JAVA_PATH}

注意位置

6，加载更新文件、验证

source profile
javac

Linux下安装MySQL

1，下载MySQL安装包

wget http://dev.mysql.com/get/mysql-community-release-el7-5.noarch.rpm

2，设置安装权限后安装

chmod a+x mysql-community-release-el7-5.noarch.rpm

rpm -ivh mysql-community-release-el7-5.noarch.rpm

3，下载mysql-community-server

yum install mysql-community-server

4，重启MySQL

service mysqld restart

5，进入MySQL数据库

mysql -u root

linux下安装Tomcat

1，下载包

wget https://downloads.apache.org/tomcat/tomcat-9/v9.0.36/bin/apache-tomcat-9.0.36.zip

2，解压包

unzip apache-tomcat-9.0.36.zip
yun install unzip

sh startup.sh
sh shutdown.sh
