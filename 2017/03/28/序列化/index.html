序列化和反序列化

Author: 陈士方 Date: 三月 28, 2017  16:21:18 Category: 笔记

序列化和反序列化的定义

序列化:就是把不可传输的对象转换为可存储或可传输的过程

反序列化:就是把在磁盘,等介质中的数据转换为对象

为什么要序列化？

1，持久化保持状态；

2，跨平台的数据交互；

json模块的使用

dumps 　　将对象序列化成bytes对象；

dump 　　 对象序列化到文件对象，就是存入文件；

loads　　 从bytes对象反序列化；

load　　 对象反序列化，从文件读取数据；

1、序列化，数据到文件，就dump；

2、反序列化，文件到数据，就load；

3、根据字符串转换就加s；

4、从文件直接转就不加s；

区别

1，load 和 loads （反序列化）
load：针对文件句柄，将json格式的字符转换为dict，从文件中读取 (将string转换为dict)

loads：针对内存对象，将string转换为dict (将string转换为dict)

2，dump 和 dumps（序列化）

dump：将dict类型转换为json字符串格式，写入到文件 （易存储）

dumps：将dict转换为string (易传输)
