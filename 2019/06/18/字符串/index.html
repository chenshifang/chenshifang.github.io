python基础-字符串

Author: 陈士方 Date: 六月 18, 2019  22:02:18 Category: python

find()

检测元素是否包含在字符串中，如果存在，则返回索引值，否则返回-1

rfind

类似于 find()函数，不过是从右边开始查找.

index()

检测 元素 是否包含在 字符串中，如果是返回开始的索引值，否则报错

count()

返回元素在字符串中出现的次数

replace()

替换字符串中的元素

split()

将字符串中的元素分割，返回一个列表

startswith

检查字符串是否是以指定元素开头, 是则返回 True，否则返回 False

endswith

检查字符串是否是以指定元素结束, 是则返回 True，否则返回 False

upper

转换字符串中的小写字母为大写

lower

转换字符串中的大写字母为小写

title

把字符串的每个单词首字母大写

capitalize

把字符串的第一个字符大写

partition

把字符串以str分割成三部分,str前，str和str后

rpartition

类似于 partition()函数,不过是从右边开始

splitlines

按照行分隔，返回一个包含各行作为元素的列表

isalpha

如果 mystr 所有字符都是字母 则返回 True,否则返回 False

isdigit

如果 mystr 只包含数字则返回 True 否则返回 False

isalnum

如果 mystr 所有字符都是字母或数字则返回 True,否则返回 False

isspace

如果 mystr 中只包含空格，则返回 True，否则返回 False

lstrip

删除 my_str 左边的空白字符

rstrip

删除 mystr 字符串末尾的空白字符

strip

删除mystr字符串两端的空白字符

join

str 中每个字符后面插入my_str,构造出一个新的字符串
